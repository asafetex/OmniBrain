# Manifest

## Missão

Manter um hub operacional de alto sinal para orientar execução diff-first com governança TRIAD e memória acumulativa.

## Regras globais

- Sem `git diff`, sem revisão.
- L3 exige Gate Codex + Gemini.
- ByteRover é memória principal; INBOX é fallback.
- Graph recebe apenas conteúdo curado.

## Contratos

- IDs estáveis por nó.
- Estrutura de nota atômica:
  - Quando usar
  - Objetivo
  - Checklist
  - Snippet mínimo
  - Validação objetiva
  - Armadilhas
  - Links relacionados
