# Context Hub

Vault Obsidian curado para roteamento de contexto, governança e reuso.

## Estrutura

- `00_HOME.md`: ponto de entrada.
- `01_MANIFEST/`: políticas globais.
- `02_GRAPH/`: skill graph por disciplina.
- `05_INBOX/`: importações brutas (fallback ByteRover).

## Regra de uso

- Abrir 2 a 5 nós relevantes por tarefa.
- Evitar carregar conteúdo excessivo.
- Promover somente conhecimento validado.
