# Project Template

Template para integrar qualquer repositório ao OmniBrain TRIAD.

## Arquivos

- `CLAUDE.md`: roteador para execução no Claude Code.
- `AGENTS.md`: contrato de revisão para Codex.
- `context-hub/README.md`: instruções para apontar para hub externo (sem duplicar vault).

## Como usar

1. Copie `CLAUDE.md` e `AGENTS.md` para a raiz do seu projeto.
2. Garanta acesso ao `context-hub/` do `omnibrain-triad` (pasta ao lado ou submodule).
3. Ajuste paths caso seu monorepo tenha estrutura diferente.
